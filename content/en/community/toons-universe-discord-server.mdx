2.  **Locate the Server Addition Tool:** On the left-hand sidebar, scroll past your current server list and click the **"+" (Plus)** icon.

## Server Rules and Community Guidelines

To maintain a healthy environment for all ages, the **toons universe discord server** enforces a strict set of rules. Roblox communities are often diverse, and the moderation team prioritizes safety and respect above all else.

*   **Respect All Members:** Harassment, hate speech, and bullying are strictly prohibited and usually result in an immediate ban.

## Troubleshooting Common Joining Issues

Occasionally, players may encounter errors when trying to access the **toons universe discord server**. These issues are usually related to account settings or server capacity.

1.  **Invalid Invite Link:** This occurs if the link has expired or been revoked. Check the official Toons Universe Roblox group for a refreshed link.
2.  **Server Limit Reached:** Discord accounts have a limit of 100 servers (or 200 with Nitro). If you cannot join, you may need to leave an old server first.
3.  **Verification Failures:** If the verification bot does not DM you, check your privacy settings to ensure "Allow direct messages from server members" is toggled on.
4.  **IP Bans:** If you have been banned from the server on a previous account, Discord's system may prevent you from joining on a new account using the same internet connection.
