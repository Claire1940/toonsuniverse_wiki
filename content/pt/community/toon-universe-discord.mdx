<Section icon={Swords} title="Regras e Etiqueta da Comunidade">

Para manter a "simbiose entre natureza e tecnologia", o **discord do toon universe** aplica um conjunto de diretrizes estritas, mas justas. Isso garante que o átrio permaneça um espaço ilimitado para todas as imaginações.

| Regra | Descrição | Consequência da Violação |
| :--- | :--- | :--- |
| **Seja Respeitoso** | Sem assédio ou discurso de ódio | Banimento permanente imediato |
| **Sem Spam** | Mantenha as mensagens relevantes para o canal | Aviso seguido de mudo temporário |
| **Mantenha-se no Tópico** | Use #off-topic para conversas fora do jogo | Exclusão de mensagem/Aviso verbal |
| **Não Negocie Dinheiro Real** | Apenas itens do jogo podem ser discutidos | Banimento permanente para RMT |
| **Use Tickets de Suporte** | Não "pingue" desenvolvedores para bugs individuais | Fechamento de ticket/Aviso |

Os moderadores usam um sistema de "Três Avisos" para infrações menores, garantindo que todos tenham a chance de aprender a cultura da comunidade. No entanto, violações graves relacionadas à segurança ou assédio são tratadas com uma política de tolerância zero.

<Callout type="warning" title="Relatando Problemas">
Se você encontrar um jogador violando essas regras, não se envolva. Use o comando `/report` ou envie uma mensagem direta a um moderador com capturas de tela para manter a paz do átrio central.
